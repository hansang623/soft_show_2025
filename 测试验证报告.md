#### 3. 大数据展示屏布局优化
**优化内容**：
- 消除右下角空置区域
- 优化高度分配：header(12vh) + main(flex:1) + footer(6vh)
- 调整字体大小和间距，确保内容适配
- 使用flexbox + grid混合布局

**布局结构**：
```css
body {
    display: flex;
    flex-direction: column;
    height: 100vh;
}

.main-header {
    min-height: 12vh;
    flex-shrink: 0;
}

.main-content {
    flex: 1;
    display: grid;
    grid-template-columns: 1fr 1fr;
    grid-template-rows: 1fr 1fr;
}

.main-footer {
    min-height: 6vh;
    flex-shrink: 0;
}
```

**验证结果**：✅ 全屏无空置，布局协调
